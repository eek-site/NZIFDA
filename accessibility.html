Accessibility Statement

Last Updated: December 2024

1. Our Commitment

The New Zealand Insoluble Fuel Disposal Agency (NZIFDA) is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards to achieve these goals.

We believe that everyone should be able to access and use our website, regardless of their abilities or the technology they use. This commitment aligns with:

  • The New Zealand Web Accessibility Standard (NZS 4121:2001)
  • The Web Content Accessibility Guidelines (WCAG) 2.1 Level AA
  • The New Zealand Human Rights Act 1993
  • International best practices for web accessibility

2. Accessibility Standards

We aim to conform to WCAG 2.1 Level AA standards. This means our website strives to be:

  • Perceivable: Information and user interface components must be presentable to users in ways they can perceive
  • Operable: User interface components and navigation must be operable
  • Understandable: Information and the operation of user interface must be understandable
  • Robust: Content must be robust enough to be interpreted by a wide variety of user agents, including assistive technologies

3. Current Accessibility Features

3.1 Keyboard Navigation

Our website can be navigated using only a keyboard. All interactive elements are accessible via keyboard shortcuts and tab navigation.

3.2 Screen Reader Compatibility

We use semantic HTML and ARIA (Accessible Rich Internet Applications) attributes to ensure compatibility with screen readers and other assistive technologies.

3.3 Text Alternatives

We provide alternative text descriptions for images and visual content to ensure information is accessible to users who cannot see images.

3.4 Color Contrast

We maintain sufficient color contrast ratios (at least 4.5:1 for normal text and 3:1 for large text) to ensure text is readable for users with visual impairments.

3.5 Responsive Design

Our website is designed to work on various devices and screen sizes, including mobile phones, tablets, and desktop computers.

3.6 Text Resizing

Users can resize text up to 200% without loss of content or functionality using browser zoom controls.

3.7 Form Labels

All form fields have associated labels to help screen reader users understand what information is required.

4. Known Limitations

While we strive to make our website fully accessible, we acknowledge that some areas may have limitations. We are actively working to address these issues:

  • Some PDF documents may not be fully accessible. We are working to provide accessible alternatives or HTML versions.
  • Some third-party content (such as embedded maps or external widgets) may not meet our accessibility standards. We are working with providers to improve this.
  • Older content may not fully meet current accessibility standards. We are gradually updating legacy content.

5. Assistive Technologies

Our website is designed to work with common assistive technologies, including:

  • Screen readers (JAWS, NVDA, VoiceOver, TalkBack)
  • Screen magnification software
  • Voice recognition software
  • Alternative input devices (switches, eye-tracking devices)
  • Browser accessibility features

6. Browser Compatibility

We test our website with the following browsers and their accessibility features:

  • Google Chrome (with accessibility extensions)
  • Mozilla Firefox (with accessibility extensions)
  • Microsoft Edge (with accessibility features)
  • Apple Safari (with VoiceOver)

When reporting accessibility issues, please include:

  • The URL of the page with the issue
  • A description of the problem
  • The assistive technology you are using (if applicable)
  • Your browser and operating system

8. Alternative Formats

If you need information from our website in an alternative format, please contact us. We can provide:

  • Large print documents
  • Audio recordings
  • Braille documents (upon request)
  • Plain language summaries
  • Other formats as needed

We will make every effort to provide alternative formats in a timely manner.

9. Ongoing Improvements

We are committed to continuously improving the accessibility of our website. Our ongoing efforts include:

  • Regular accessibility audits and testing
  • Staff training on accessibility best practices
  • User testing with people with disabilities
  • Staying current with accessibility standards and guidelines
  • Addressing identified barriers promptly

10. Third-Party Content

Some content on our website is provided by third parties (such as links to the EPA website). While we strive to link to accessible content, we cannot guarantee the accessibility of all third-party websites. If you encounter accessibility issues with third-party content, please let us know, and we will work to find alternatives.

11. Compliance Status

Current Status: We are actively working toward full WCAG 2.1 Level AA compliance. Our website is partially conformant with WCAG 2.1 Level AA. Some content may not fully conform to the accessibility standard.

We regularly assess our website's accessibility and make improvements based on:

  • Automated accessibility testing tools
  • Manual testing by accessibility experts
  • User feedback from people with disabilities
  • Internal accessibility reviews

12. Enforcement

If you believe we have not responded to your accessibility concern in a satisfactory manner, you may file a complaint with:

  • New Zealand Human Rights Commission: www.hrc.co.nz
  • Office of the Ombudsman: www.ombudsman.parliament.nz

13. Updates to This Statement

We will update this Accessibility Statement as we make improvements to our website's accessibility. The "Last Updated" date at the top of this page indicates when this statement was last revised.
